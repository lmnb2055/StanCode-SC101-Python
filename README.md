## 💻 Projects & Assignments Overview

### 🎮 Game Design & Graphics
* **Assignment 0: My Drawing**
    * A creative introduction to custom class creation and digital art using Python.
* **Assignment 1: Bouncing Ball & Draw Line**
    * Focuses on physics simulation and mouse event handling.
* **Assignment 2: Breakout Game** (`breakout.py`, `breakoutgraphics.py`)
    * A fully functional interactive arcade game.
    * Concepts: **Object-Oriented Programming (OOP)**, Event-driven programming, and GUI management.

### 📊 Data Visualization & Image Processing
* **Assignment 3: stanCodoshop**
    * An advanced image processing tool that uses statistical filters (like the median) to remove unwanted objects/noise from photos.
* **Assignment 4: Baby Names Explorer** (`babynames.py`, `webcrawler.py`)
    * A large-scale data visualization project.
    * Includes a **Web Crawler** and a GUI to track name popularity trends over decades.

### 🧠 Algorithms & Problem Solving
* **Assignment 5: Recursive Algorithms** (`anagram.py`, `sierpinski.py`)
    * Implementing complex patterns like the **Sierpinski Triangle** and **Anagram** generation using **Recursion**.
* **Assignment 6: Boggle Game & Linked Lists** (`boggle.py`, `add2.py`)
    * A word search game built with **Recursive Backtracking**.
    * Introduction to **Linked List** operations and data structure manipulation.
